**让语言模型拥有“表情”和“灵魂感”的交互尝试**
**An attempt to give LLM-driven characters expression, responsiveness, and presence**

* 基于对话内容理解情绪与语义
  Understands emotion and conversational intent from dialogue
* 驱动 Live2D 参数与表情变化
  Controls Live2D parameters and expression states in real time
* 支持 WebSocket 通信、多模型加载与参数过渡
  Supports WebSocket communication, multi-model loading, and smooth parameter transitions
* 体现了你在 **AI + 实时交互 + 虚拟角色系统** 方向上的辨识度
  Highlights your strengths in **AI + real-time interaction + virtual character systems**

  Built with **Spring Boot + MyBatis + MySQL + LLM API**
* 包含 AI 规划、学习记录、统计分析与辅助功能
  Includes AI-based planning, study tracking, analytics, and assistant features
* 展示了从后端到数据库再到 AI 集成的完整工程实践
  Demonstrates full workflow capability from backend engineering to database design and AI integration
